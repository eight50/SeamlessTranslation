# SeamlessTranslation (Ongoing Project)

ウィジェット上で完結する音声翻訳アプリ  
ウィジェットから録音を開始し、翻訳結果を表示する  
VoiceToTextはVosk、翻訳はDeepLAPIで行う

## 学ぶこと
Clean Architecture + MVVM  
Kotlin  
ForegroundService  
Glance  
Compose  
Retrofit  
できたらCoroutine  
Vosk : Speech Recognition(Apache License 2.0)
